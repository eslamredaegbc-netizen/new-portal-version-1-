# منصة الرصد الإعلامي لقطاع الإعلام

الجهة المالكة: قطاع التطوير التقني  
الحقوق: جميع الحقوق محفوظة لقطاع التطوير التقني

هذه هي النسخة الأساسية للنظام ببنية `JavaScript + FastAPI` كما طُلب للمشروع، مع قاعدة بيانات داخلية `SQLite`، وتقارير فعلية، ومحرك رصد متعدد المصادر، ومساعد داخلي يعتمد فقط على بيانات النظام.

## التشغيل الأساسي الموصى به

1. `pip install -r requirements.txt`
2. `uvicorn app:app --host 0.0.0.0 --port 8000`

ثم افتح:

- `http://127.0.0.1:8000`

## تشغيل بديل عبر Streamlit

إذا أردت نسخة تشغيل بديلة عبر Streamlit:

1. `pip install -r requirements.txt`
2. `streamlit run streamlit_app.py`

مهم:

- `app.py` هو الآن نقطة التشغيل الأساسية للواجهة `JavaScript + FastAPI`
- `streamlit_app.py` نسخة بديلة وليست المسار الرئيسي الموصى به لهذا المشروع

## مزودات البحث والدعم الاختياري

### Google Web Search

لتفعيل Google Web وGoogle Images:

- `GOOGLE_SEARCH_API_KEY`
- `GOOGLE_SEARCH_ENGINE_ID`

مهم:

- بحسب توثيق Google الرسمي الحالي، خدمة `Custom Search JSON API` متاحة فقط للعملاء الحاليين حتى `January 1, 2027` وليست متاحة للعملاء الجدد
- لهذا السبب يعتمد النظام على Google عند توفره، ويستخدم مزودات وفوالباك أخرى عند غيابه

### YouTube Data API

لتفعيل بحث YouTube الرسمي مباشرة:

- `YOUTUBE_API_KEY`

عند عدم ضبطه، يستخدم النظام فوالباك بحث أخرى لالتقاط نتائج YouTube العامة.
