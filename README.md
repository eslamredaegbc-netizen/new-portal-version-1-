# منصة الرصد الإعلامي لقطاع الإعلام

الجهة المالكة: قطاع التطوير التقني  
الحقوق: جميع الحقوق محفوظة لقطاع التطوير التقني

هذه النسخة تدعم مسارين للتشغيل:
- مسار `Streamlit` للنشر على `Streamlit Cloud`
- مسار `FastAPI + JavaScript` للتشغيل الويب المباشر

## التشغيل الأساسي على Streamlit

1. `pip install -r requirements.txt`
2. `streamlit run app.py`

هذا هو المسار الصحيح للنشر على `Streamlit Cloud`، كما أن `app.py` هو الملف الذي يجب اختياره كـ `Main file`.

## تشغيل واجهة FastAPI + JavaScript

1. `pip install -r requirements.txt`
2. `uvicorn app:app --host 0.0.0.0 --port 8000`

ثم افتح:

- `http://127.0.0.1:8000`

## ملاحظات التشغيل

- `app.py` أصبح متوافقًا مع `Streamlit Cloud`، وفي نفس الوقت يكشف كائن `FastAPI` للتشغيل عبر `uvicorn`
- `streamlit_app.py` يحتوي واجهة Streamlit الكاملة، و`app.py` يستخدمها كنقطة دخول للنشر
- `api_app.py` يحتوي واجهات الـAPI الخاصة بمسار `FastAPI + JavaScript`
